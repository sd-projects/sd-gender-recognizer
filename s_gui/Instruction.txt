В открываемой программе должна быть функция "Algorithm()", в которой записан алгоритм программы
Так же должны быть заданы настройки ввода и вывода
Установка настроек описывается ниже

Ввод настроек ввода производится через присваивание переменной "imp" массива
Первым элементом массива является тип вводимых данных
Далее в массиве идут дополнительные настройки ввода (конкретнее в примерах)

Примеры ввода:

imp = ["file", "P. F. (*.py;*.py3)"]
# Ввод файла одной кнопкой, во втором элементе указываются расширения файла

imp = ["two_files", "P. F. (*.py;*.py3);;JPEG (*.jpg;*.jpeg)", "JPEG (*.jpg;*.jpeg)"]
# Ввод двух файлов, двумя кнопками, во втором и третьем элементах указываются расширения файлов

imp = ["files", "P. F. (*.py;*.py3);;JPEG (*.jpg;*.jpeg)"]
# Ввод нескольких файлов одной кнопкой, во втором элементе указываются расширения файлов

ipm = ["str"]
# Ввод строки, с помощью одного поля

imp = ["two_str"]
# Ввод двух строк, с помощью двух полей

imp = ["int"]
# Ввод числа, с помощью одного поля

imp = ["two_int"]
# Ввод двух чисел, с помощью двух полей


Примеры вывода:
rlt = file
# Программа записывает результат в results.txt

rlt = [str]
# Программа выводит в графический интерфейс строку текста
# P.S. Если длинна выводимой строки более 30 символов, данные записываются в файл results.txt
